# README

Este projeto foi criado para rodar em uma servidor ubuntu 24.04.2 LTS (GNU/Linux 6.8.0-51-generic x86_64)

Para rodar a base deste projeto, você precisa ter as seguintes tecnologias e versões:

Ruby 3.2.3 
Você pode usar o Rbenv para fazer a instalação do ruby, neste caso, especificando a versão 3.2.3.
(RBENV Doc: https://github.com/rbenv/rbenv)

Rails 8.0.1
(Rails Doc: https://github.com/rails/rails/tree/v8.0.1)

Mysql2:
(Doc: https://sidorares.github.io/node-mysql2/pt-BR/docs/documentation)
